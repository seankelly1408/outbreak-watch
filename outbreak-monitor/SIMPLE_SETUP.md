# 🚀 SIMPLEST SETUP GUIDE - 8 Steps to Live & Ranked

## Step 1: Create GitHub (1 minute)
1. Go to **github.com**
2. Sign up for free
3. Done ✓

---

## Step 2: Create Repository (1 minute)
1. Go to **github.com/new**
2. Name it: `outbreak-watch`
3. Make it Public
4. Click "Create repository"
5. Done ✓

---

## Step 3: Upload Files (3 minutes)
1. Download ALL files from your outputs folder
2. On your GitHub repo page, click "Add file" → "Upload files"
3. **Drag and drop ALL files** (including the new ones):
   - App.jsx, App.css, server.js, scraper.py
   - package.json, vite.config.js, Dockerfile
   - **index-seo.html** (IMPORTANT - this is the SEO version!)
   - robots.txt, sitemap.xml
   - .gitignore, .env.example, main.jsx

4. Scroll down → Click "Commit changes"
5. Done ✓

---

## Step 4: Rename index-seo.html to index.html
1. On your GitHub repo, click on "index-seo.html"
2. Click the pencil icon to edit
3. At the top, change filename from `index-seo.html` to `index.html`
4. Scroll down → Click "Commit changes"
5. Done ✓

---

## Step 5: Deploy on Railway (2 minutes)
1. Go to **https://railway.app**
2. Click "Start a new project"
3. Click "Deploy from GitHub repo"
4. Click "Authorize with GitHub" and approve
5. Select your `outbreak-watch` repo
6. Click "Deploy now"
7. **WAIT 1-2 minutes for deployment to finish** ⏳
8. Done ✓

---

## Step 6: Get Your Live URL (30 seconds)
1. Railway will show you a URL like: `outbreak-watch.up.railway.app`
2. **Click it to test** - Your site should be live!
3. Write down this URL
4. Done ✓

---

## Step 7: Tell Google About Your Site (5 minutes)
This is the SECRET to ranking for "hantavirus outbreak" ✨

1. Go to **google.com/search-console** 
2. Click "Start Now"
3. Paste your URL: `https://outbreak-watch.up.railway.app`
4. Follow Google's verification (they'll ask you to add a DNS record or HTML file)
   - *Easiest: Click "Add record to DNS provider"*
   - *Or: Download HTML file and upload to your site*
5. Click "Submit sitemap": `https://outbreak-watch.up.railway.app/sitemap.xml`
6. **DONE** - Google is now indexing your site! 🎉
